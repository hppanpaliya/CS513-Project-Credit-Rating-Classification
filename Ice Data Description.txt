Data: ICE Bond Data 
- Intercontinental Bond Data Index which tracks $100 trillion in global market debt
- HY Index (high yield bonds are bonds rated BB+ and lower)
- Time-series data for September 1, 2023, this has ~ 1800 rows, each row representing a bond with all the attributes and a specific credit rating
- Idea: Use features to classify the Rating column for each bond to build an accurate model for predicting Bond ratings based on bond characteristics.
- Glossary of features: https://www.ice.com/publicdocs/data/Bond_Index_Methodologies.pdf (page 73)
- Specific features pulled from data are as follows:

- Steps:
1. Data Cleaning (Dealing with null values, null columns)
2. Feature Selection (Determining best subset of features from full list)
3. Feature Extraction (From selected features perform dimensionality reduction to reduce multi-colinearity and other issues)
4. Basic Models (Naive Bayes, KNN, CART)
5. Advanced Models (Logistic Regression, Neural Network, XGBoost Classifier)
6. Evaluate models using different metrics and draw conclusions
